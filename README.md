# Advanced Hydrologic Analysis - HW1
本儲存庫包含高等水文分析第一次作業之Python程式碼
## 作業內容
* Q1: 資料提取與探索性分析 (Data Extraction & Exploratory Analysis)
* Q2: 機率分佈擬合 (Probability Distribution Fitting)
* Q3: 適配度檢定 - 卡方檢定 (Goodness-of-Fit - Chi-Square Test)
* Q4: 適配度檢定 - Q-Q 圖與誤差分析 (Q-Q Plot & Error Analysis)
* Q5: 降雨資料之蒙地卡羅模擬 (Monte Carlo Simulation of Rainfall Data)
* Q6: 重現期水準估計 (Return Level Estimation)
